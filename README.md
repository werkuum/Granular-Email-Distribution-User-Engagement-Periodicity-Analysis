# Granular-Email-Distribution-User-Engagement-Periodicity-Analysis
Цей технічний кейс присвячений аналізу інтенсивності емейл-маркетингу на рівні окремих користувачів. Основна мета — визначити частку «уваги» (обсягу трафіку), яку отримує кожен акаунт у загальній масі розсилок, та зафіксувати часові межі їхньої активності в межах кожного місяця.
